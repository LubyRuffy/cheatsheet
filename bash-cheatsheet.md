# 基础知识
* 变量
```
NAME="John"
echo $NAME
echo "$NAME"
echo "${NAME}!"
```

* 字符串引号
```
NAME="John"
echo "Hi $NAME"  #=> Hi John
echo 'Hi $NAME'  #=> Hi $NAME
```

* 函数
```
get_name() {
  echo "John"
}

echo "You are $(get_name)"
```
